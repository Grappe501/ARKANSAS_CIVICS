# Full app.js replacement

This package contains a full upgraded replacement for:
`apps/phase-09-course-dashboard/app.js`

## What it preserves
- Search
- Bookmarks
- Resume last
- Reset progress
- Course/chapter/segment navigation
- Stats
- Markdown lesson rendering

## What it adds
- Activation lesson support
- Action checkbox persistence
- Mission logging
- Reflection saving
- Next lesson button support
- Dual-format compatibility (`PHASE095_CONTENT` and `PHASE12_CONTENT`)
