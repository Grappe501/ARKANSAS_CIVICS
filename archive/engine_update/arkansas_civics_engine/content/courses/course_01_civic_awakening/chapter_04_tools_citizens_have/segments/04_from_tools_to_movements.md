## Canonical Body

Civic tools matter, but tools alone do not create change. People do. A ballot process, a public meeting, a voting system reform, or a local board seat only matters when citizens understand how to use it strategically. That is where movements begin: not with technique alone, but with shared pain, shared story, shared purpose, and organized follow-through.

Course 1 ends here by shifting the reader from awareness to agency. The next question is no longer only what is broken. It is what kind of movement could grow from the tools already in the people's hands.

## Workbook Prompt

Choose one issue you care about. Which tool fits it best: relationship-building, local board advocacy, direct democracy, public storytelling, or coalition organizing?

## Workshop Activity

Groups choose a hypothetical issue and build a first-draft movement pathway from problem to institution to tactic.
