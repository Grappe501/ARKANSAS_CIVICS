Phase 13

AI Lesson Generator Helper

This helper builds prompts and shows the activation schema. Generation runs through the Python script in your repo.

Activation schema

{
  "title": "Lesson title",
  "hook": "Short opening hook",
  "reality": "Clear civic explanation",
  "actions": ["Action 1", "Action 2", "Action 3"],
  "reflection": "Reflection prompt",
  "next_step": "Next lesson or next move",
  "tags": ["tag1", "tag2"]
}

Recommended commands

python scripts/generate_activation_lessons_ai.py --course course_01_civic_awakening --chapter chapter_01 --mode template
python scripts/generate_activation_lessons_ai.py --course course_01_civic_awakening --chapter chapter_01 --mode openai
python scripts/generate_activation_lessons_ai.py --all --mode template

OpenAI mode requirements

Set OPENAI_API_KEY and install the OpenAI Python package in your local environment.
