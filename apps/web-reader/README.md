# Web Reader

Recommended deployment target: Netlify.

## Why tie website + Articulate together?

Yes. The website should be the deep-learning layer and source archive.

Articulate should do:
- guided pathway
- simulations
- quizzes
- progressive release

Website should do:
- long-form reading
- story archive
- county data explorer
- source library
- chapter extensions
- future spawned tracks

## Linking pattern

Each Articulate lesson should include:
- `Read deeper on the site`
- `Explore the county data`
- `See the Arkansas timeline`
- `Open the source story archive`

Each website chapter should include:
- `Take the guided lesson`
- `Try the simulation`
- `Save notes`
- `Continue to workshop mode`
