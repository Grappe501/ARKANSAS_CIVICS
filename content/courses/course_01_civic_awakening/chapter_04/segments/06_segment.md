# Voting Systems

## Book Narrative

Not all voting systems operate the same way.

Different systems produce different political outcomes.

Some systems encourage majority representation.

Others allow minority voices greater influence.

Understanding voting systems helps citizens evaluate how democratic institutions function.

## Course Activity

Learners compare different voting systems.

## Workbook Exercise

Which voting system seems most fair to you?

## Workshop Activity

Simulate elections using different voting methods.
